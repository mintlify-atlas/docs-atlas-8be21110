---
title: "Quickstart"
description: "Connect to PostgreSQL from Java in minutes using the PgJDBC driver."
---

This guide gets you from zero to a working PostgreSQL connection in Java. You will add the dependency, establish a connection, and run your first query.

## Prerequisites

- Java 8 or higher
- A running PostgreSQL server (8.4+)
- Maven or Gradle

## Steps

<Steps>
  <Step title="Add the dependency">
    Add PgJDBC to your project from Maven Central. No additional repositories are needed.

    <CodeGroup>

    ```xml Maven (pom.xml)
    <dependency>
      <groupId>org.postgresql</groupId>
      <artifactId>postgresql</artifactId>
      <version>42.7.10</version>
    </dependency>
    ```

    ```kotlin Gradle (build.gradle.kts)
    implementation("org.postgresql:postgresql:42.7.10")
    ```

    ```groovy Gradle (build.gradle)
    implementation 'org.postgresql:postgresql:42.7.10'
    ```

    </CodeGroup>

    <Note>
      The driver registers itself automatically via the Java Service Provider mechanism. You do not need to call `Class.forName("org.postgresql.Driver")`.
    </Note>
  </Step>

  <Step title="Import the JDBC API">
    Import the standard `java.sql` package in your source file. Only import `org.postgresql` packages if you need PostgreSQL-specific extensions.

    ```java
    import java.sql.*;
    ```
  </Step>

  <Step title="Connect to your database">
    Use `DriverManager.getConnection()` with a PostgreSQL JDBC URL.

    <Tabs>
      <Tab title="URL with credentials">
        ```java
        String url = "jdbc:postgresql://localhost:5432/mydb";
        Connection conn = DriverManager.getConnection(url, "myuser", "mypassword");
        ```
      </Tab>
      <Tab title="Properties object">
        ```java
        String url = "jdbc:postgresql://localhost:5432/mydb";
        Properties props = new Properties();
        props.setProperty("user", "myuser");
        props.setProperty("password", "mypassword");
        props.setProperty("ssl", "true");
        Connection conn = DriverManager.getConnection(url, props);
        ```
      </Tab>
      <Tab title="All-in-URL">
        ```java
        String url = "jdbc:postgresql://localhost:5432/mydb?user=myuser&password=mypassword";
        Connection conn = DriverManager.getConnection(url);
        ```
      </Tab>
    </Tabs>

    <Warning>
      If your password contains special URL characters (e.g., `@`, `#`, `&`), use the Properties object approach or percent-encode the value in the URL.
    </Warning>
  </Step>

  <Step title="Execute a query">
    Use `Statement` for one-off queries, or `PreparedStatement` for parameterized queries.

    <Tabs>
      <Tab title="Simple query">
        ```java
        Statement st = conn.createStatement();
        ResultSet rs = st.executeQuery("SELECT version()");
        while (rs.next()) {
            System.out.println("PostgreSQL version: " + rs.getString(1));
        }
        rs.close();
        st.close();
        ```
      </Tab>
      <Tab title="Parameterized query">
        ```java
        int targetId = 42;
        PreparedStatement ps = conn.prepareStatement(
            "SELECT name FROM users WHERE id = ?"
        );
        ps.setInt(1, targetId);
        ResultSet rs = ps.executeQuery();
        while (rs.next()) {
            System.out.println("User: " + rs.getString("name"));
        }
        rs.close();
        ps.close();
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Close your resources">
    Always close `ResultSet`, `Statement`, and `Connection` when done. Use try-with-resources to ensure cleanup even on exceptions.

    ```java
    try (Connection conn = DriverManager.getConnection(url, "myuser", "mypassword");
         PreparedStatement ps = conn.prepareStatement("SELECT version()");
         ResultSet rs = ps.executeQuery()) {

        if (rs.next()) {
            System.out.println("Connected: " + rs.getString(1));
        }
    }
    // conn, ps, and rs are automatically closed here
    ```
  </Step>
</Steps>

## Complete example

Here is a full working program you can run:

```java QuickstartExample.java
import java.sql.*;
import java.util.Properties;

public class QuickstartExample {
    public static void main(String[] args) throws SQLException {
        String url = "jdbc:postgresql://localhost:5432/mydb";
        Properties props = new Properties();
        props.setProperty("user", "myuser");
        props.setProperty("password", "mypassword");

        try (Connection conn = DriverManager.getConnection(url, props)) {
            System.out.println("Connected to PostgreSQL!");

            // Execute a simple query
            try (Statement st = conn.createStatement();
                 ResultSet rs = st.executeQuery("SELECT version()")) {
                if (rs.next()) {
                    System.out.println("Server: " + rs.getString(1));
                }
            }

            // Parameterized insert
            try (PreparedStatement ps = conn.prepareStatement(
                    "INSERT INTO test_table (name, value) VALUES (?, ?)")) {
                ps.setString(1, "hello");
                ps.setInt(2, 100);
                int rows = ps.executeUpdate();
                System.out.println("Inserted " + rows + " row(s)");
            }
        }
    }
}
```
