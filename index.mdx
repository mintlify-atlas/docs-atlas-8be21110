The PostgreSQL JDBC Driver (PgJDBC) is an open-source, pure Java Type 4 driver that lets Java programs connect to a PostgreSQL database using standard, database-independent JDBC code. It communicates directly in the PostgreSQL native network protocol — no native libraries required.

## Get started in 3 steps

<Steps>
  <Step title="Add the dependency">
    Add PgJDBC to your Maven or Gradle project from Maven Central:

    ```xml pom.xml
    <dependency>
      <groupId>org.postgresql</groupId>
      <artifactId>postgresql</artifactId>
      <version>42.7.10</version>
    </dependency>
    ```
  </Step>
  <Step title="Connect to your database">
    Use the standard JDBC `DriverManager` with a PostgreSQL connection URL:

    ```java
    import java.sql.*;

    Connection conn = DriverManager.getConnection(
        "jdbc:postgresql://localhost:5432/mydb",
        "myuser",
        "mypassword"
    );
    ```
  </Step>
  <Step title="Run your first query">
    Execute SQL and process results using standard JDBC interfaces:

    ```java
    try (Statement st = conn.createStatement();
         ResultSet rs = st.executeQuery("SELECT version()")) {
        if (rs.next()) {
            System.out.println("Connected to: " + rs.getString(1));
        }
    }
    ```
  </Step>
</Steps>

## Supported versions

| Component | Supported versions |
|---|---|
| PostgreSQL | 8.4 and higher (9.1+ recommended) |
| Java | 8 (JDBC 4.2) and above |
| JDBC spec | 4.2 |

<Note>
  PgJDBC versions since 42.8.0 are not guaranteed to work with PostgreSQL older than 9.1. For the best experience, use PostgreSQL 9.1 or later.
</Note>
